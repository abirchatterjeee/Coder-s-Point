6.PROGRAM TO FIND FACTORIAL OF THE NUMBER:

#include<stdio.h>

int main()

{

int i,n,f=1;

printf("Enter the number whose factorial be found: ");

scanf("%d",&n);

for(i=1;i<=n;i++)

{

f=f*i;

}

printf("Factorial of number: %d",f);

}

OUTPUT :

Enter the number whose factorial be found: 5

Factorial of number: 120
