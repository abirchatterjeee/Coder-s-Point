30.PROGRAM TO CHECK IF THE NUMBER IS DUCK NUMBER OR NOT : (A number is said to be a duck number if zero is present in that number)

#include<stdio.h>

int main()

{

int n,a,n1;

printf("Enter the number:");

scanf("%d",&n);

n1=n;

while(n>0)

{

a=n%10;

n=n/10;

if(a==0)

{

printf("%d is duck number",n1);

}

}

}

OUTPUT :

Enter the number:5012

5012 is duck number
